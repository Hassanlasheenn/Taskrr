# TaskHub Application

A full-stack task management application that enables users to create, organize, and track their tasks with features like user authentication, task assignment, priority management, categories, and real-time notifications.

## What It Does

This application serves as a collaborative task management system where users can:
- Create, update, and delete todos with priorities and categories
- Assign tasks to other users for collaboration
- Receive notifications when tasks are assigned to them
- Manage user profiles and roles (User/Admin)
- Access an admin panel for user and system management

## Tech Stack

### Backend
- **FastAPI** - Python web framework
- **SQLAlchemy** - ORM for database operations
- **Azure SQL Edge** - SQL Server database
- **JWT** - Authentication tokens
- **Argon2** - Password hashing
- **Docker** - Database containerization

### Frontend
- **Angular 19** - Frontend framework
- **TypeScript** - Programming language
- **Bootstrap 5** - UI styling
- **RxJS** - Reactive state management
